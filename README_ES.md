# pi-opensim-upperlimb

Proyecto de investigación sobre simulación musculoesquelética de miembro superior con OpenSim. El objetivo es estudiar cómo modelar la interacción entre un brazo humano y un dispositivo externo (exoesqueleto o end-effector) que ejerce fuerzas sobre el sistema.

El contenido está organizado en issues de GitHub, uno por bloque temático:

- **#1 – Objetivos**
- **#2 – Pipeline y módulos de OpenSim**
- **#3 – OpenSim + miembro superior + fuerzas externas: revisión de literatura**
- **#4 – Modelado de exoesqueletos y end-effectors**
- **#5 – API de Python de OpenSim**
- **#6 – MuJoCo y herramientas alternativas**

**Herramientas:** OpenSim · Python · MuJoCo · AnyBody
