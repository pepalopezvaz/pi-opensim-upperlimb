# pi-opensim-upperlimb

Research project on musculoskeletal simulation of the upper limb using OpenSim. The goal is to study how to model the interaction between a human arm and an external device (exoskeleton or end-effector) that exerts forces on the system.

The project content is organized as GitHub issues, one per topic block:

- **#1 – Objectives**
- **#2 – OpenSim pipeline and modules**
- **#3 – OpenSim + upper limb + external forces: literature review**
- **#4 – Exoskeleton and end-effector modeling**
- **#5 – OpenSim Python API**
- **#6 – MuJoCo and alternative tools**

**Tools:** OpenSim · Python · MuJoCo · AnyBody
